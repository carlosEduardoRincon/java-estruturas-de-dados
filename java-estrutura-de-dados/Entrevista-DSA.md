### Entrevista DSA

- O que os entrevistadores buscam na hora de codar?
  - Profundidade profissional;
  - Capacidade de quebrar problemas em problemas menores;
  - Análise de tradeoffs;
  - Lógica;
  - Tomada de decisões;
  - Lidar com edge cases;
  - Traduzir requisito em código;
  - Lidar com pressão;
  - Comunicação (+);
  - Colaboração (+);

- O que não é esperado?
  - Corretude;

- Múltiplas resoluções:
  - Força bruta: de qualquer maneira que pensar, mesmo que não seja a maneira ótima/ideal, porém é a mais rápida de desenvolver (200 itens);
  - Resolvido para entrevista: boa o suficiente / razoável;
  - Otimizado: solução ideal, resposta acadêmica seguindo os padrões;

- Preparo específico para entrevistas:
  - Focar uma estrutura de dados por vez;
  - Prática deliberada: algo no limite da capacidade de aprendizado (vejas as soluções e siga para a próxima questão);
  - Repetição espaçada: refazer em determinados tempos os conteúdos;
